# Arca-de-madera-

Arca de Madera es un proyecto de tienda de muebles y decoracion en madera.
El enfoque es vender piezas funcionales y de diseno con identidad artesanal,
apoyadas por un fabricante mayorista de Ocotlan, Jalisco, con capacidad de
produccion a la medida y calidad boutique.

La operacion comercial estara enfocada en Cancun, Quintana Roo.

## Objetivo de este repositorio

Este repositorio concentra las bases para construir un negocio rentable y
escalable, usando el modelo de los 9 bloques de negocio.

## Estructura inicial

- `docs/00-identidad-corporativa.md`: identidad de marca, tono, posicionamiento y activos base.
- `docs/01-modelo-negocio-9-bloques.md`: definicion operativa de los 9 bloques.
- `docs/02-plan-45-dias-acelerado.md`: plan de implementacion acelerado por fases.
- `docs/03-kpis-y-control.md`: tablero de indicadores y control de gestion.
- `data/business-canvas.json`: version estructurada del modelo para usar en apps.

## Siguiente paso recomendado

1. Aprobar identidad corporativa y guia visual.
2. Validar propuesta de valor con 10 clientes potenciales en Cancun.
3. Iniciar ejecucion del plan acelerado de 45 dias con seguimiento semanal.
